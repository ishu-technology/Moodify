Mood-based MP3 player built using Python.

## Features:
- Login & Signup
- Mood Selection
- Music Player
- Integrated Spotify playlists for different moods (clicking the mood button opens the related Spotify playlists)

## Tech Stack:
- Python
- Tkinter
- SQLite

## How to Run:
1. Install Python
2. Run main.py
